Add Environment Variables

  1. To add Environment variables, click on Configuration under the Releases tab.

  1. At the bottom of the page, you will see Environment Settings

  1. Click on the +Create button.

  1. Enter the details of the environment setting.

  1. You may input text value or numerical value. Enter the name of the field. Describe the field to provide more information. And add a placeholder value. Placeholder value gets displayed when the input is empty. Click on Required to make it a mandatory field. After entering the details click on Save and Continue.

  1. Click on the Default Data Values tab. Click on to input Add a Default Value to enter the default value of the field.

  1. Enter the value and click on Save and Continue.

  1. The entry gets added in theEnvironment Settings. You may access the field in the instance of the schema.
