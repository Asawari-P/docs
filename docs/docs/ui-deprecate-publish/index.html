Publish / Deprecate Release

  1. There may be three types of releases; Draft, Deprecated, Published. You may see these different types as highlighted.

  1. You may change the release status under the Settingsof the Versions. The initial status of the release is Draft status. You may Publish the release if the data model has been entered in your schema. Click on Published to change the status.

  1. Click on Yes, Continue to publish the release. Hypi will create the Tables once the release gets published. You will not be able to modify existing tables/ fields after this. But you may add new fields or tables.

  1. The status of the release changes to Published.

  1. If a release is no longer supported, you can change the status to deprecated. Click on Yes, Continue to change the status to deprecated.

  1. If the status is Published, you can see the change in the Table under the Visual Builder. The fields of the table become Read-Only.
